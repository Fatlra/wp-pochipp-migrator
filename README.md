# wp-pochipp-migrator 🚀

Python-based automation tool for migrating WordPress shortcodes.
It extracts ASINs using Regex and scrapes Amazon product titles with BeautifulSoup to streamline the transition between plugins like WPAP and Pochipp.

## ✨ Features
- **Regex Extraction**: Robust ASIN extraction handles both single and double quotes.
- **Automated Scraping**: Fetches product titles directly from Amazon to fill in missing metadata.
- **Error Resilience**: Designed to skip broken links and keep processing thousands of entries.

## 🛠️ Tech Stack
- **Python 3.12**
- **BeautifulSoup4 / Requests**: Web Scraping
- **Regular Expressions (re)**: Text Processing
